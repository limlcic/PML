Terminology of regress output

In a regress task, each instance with response lable in testing data would get a regression outcome .
Assuming the data has instances, the derivations would come out by using and :

Name Description Formula
CC Correlation Coefficient
RMSE Root Mean Squared Error
MAE Mean Absolute Error
RAE Relative Absolute Error
RRSE Root Relative Squared Error

Reference:
CC: http://www.jerrydallal.com/LHSP/correg.htm
RMSE: http://en.wikipedia.org/wiki/Mean_squared_error
MAE: http://en.wikipedia.org/wiki/Mean_absolute_error
RAE, RRSE: http://www.saedsayad.com/model_evaluation_r.htm
